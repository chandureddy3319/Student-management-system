# Student Records Management System

A professional, responsive web application for managing student records, built for MCA Advanced Web Technologies practical assessment.

## Features
- User registration and login (PHP, MySQL)
- View, search, update, and delete student records (AJAX, PHP, MySQL)
- Initial data loaded from JavaScript (JSON)
- Professional UI with animations, Google Fonts, and responsive design
- All CRUD operations via AJAX

## Folder Structure
- `/css` - Stylesheets
- `/js` - JavaScript files (logic and initial data)
- `/images` - Student profile images
- `/php` - PHP backend scripts
- `/sql` - SQL database setup

## Setup Instructions
1. Import the SQL file from `/sql/student_management.sql` into your MySQL server (e.g., via phpMyAdmin).
2. Place the project folder in your XAMPP `htdocs` directory.
3. Update database credentials in `php/db.php` if needed.
4. Start Apache and MySQL from XAMPP control panel.
5. Access the app at `http://localhost/loginform/`.
